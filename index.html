GPT-ME: Arquitetura de IA com Memória Esferocêntrica

GPT-ME (Esferocentric Memory GPT) é uma arquitetura simbólica inspirada pela Teoria da Memória Esferocêntrica.

🔍 Principais Inovações

  • Processamento simbólico com reforço persistente
  • Ativação contextual por geometria relacional
  • Explicabilidade completa por rotas de ativação
  • Protótipo funcional escrito em Rust

📦 Estrutura do Projeto

  • gpt_me_prototipo.rs: Código-fonte da IA simbólica
  • Cargo.toml: Arquivo de configuração do projeto Rust
  • README.md: Descrição técnica e instruções

🚀 Execução

cargo run
